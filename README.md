# Hangman Game

A simple text-based Hangman Game built using Python.

## Features

* Random word selection
* 6 incorrect guesses allowed
* User-friendly console interface
* Win/Lose conditions

## Technologies Used

* Python

## How to Run

```bash
python hangman.py
```

## Concepts Used

* Random Module
* Loops
* Conditional Statements
* Lists and Strings
* User Input Handling
